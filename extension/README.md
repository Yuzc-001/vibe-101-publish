# Chrome Extension (V1)

目标：把当前网页中的选中文本或页面内容发送到 `vibe-101-publish` 工作台。

## 功能
- 右键菜单：发送选中文本到工作台
- Popup：
  - 发送选中文本
  - 抓取当前页内容
  - 目标切换：本地 / 线上

## 加载方式
1. 打开 Chrome -> `chrome://extensions/`
2. 开启开发者模式
3. `加载已解压的扩展程序`
4. 选择本目录 `extension/`

## 说明
- 本地目标默认：`http://127.0.0.1:5173/`
- 线上目标默认：`https://yuzc-001.github.io/vibe-101-publish/`
- 第一期先做轻量注入，不做复杂正文清洗
- 数据通过 `chrome.storage.local` -> 工作台读取桥接

## 手测建议
1. 在任意网页选中文字
2. 右键点击 `发送到 vibe-101-publish`
3. 确认工作台自动打开并写入标题 / 来源 / 内容
4. 再测试 popup 的 `抓取当前页内容`
5. 再测试 `仅打开工作台`，确认插件壳子独立可用
6. 分别验证本地目标与线上目标

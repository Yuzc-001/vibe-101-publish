# vibe-101-publish
## 把“排版工具”做成“可发布的内容产品”

vibe-101-publish 不是只会把 Markdown 渲染漂亮的编辑器。  
它是一条完整的内容发布链路：从写作、粘贴、排版、预览，到公众号与多格式导出，最终目标是“发得出去，而且发得稳定”。

## 我们在解决什么问题

很多内容工具在编辑区好看，但一到发布环节就崩：

- 公众号粘贴后样式走样
- 图片和表格结构错位
- 公式在目标平台显示异常
- 多端预览与真实发布效果不一致

我们把这些“最后一公里问题”当成主战场，而不是附加项。

## 产品理念

- 发布一致性优先：编辑体验必须对最终发布负责
- 质量可控优先：主题、文字节奏、代码、公式都可被精细调节
- 微信兼容优先：针对公众号链路持续优化
- 工程可持续优先：功能可迭代、可验证、可维护

## 核心能力

### 1) 一键公众号发布

- 复制到公众号链路深度兼容
- 外链图片自动处理
- 列表、表格、图文结构稳定输出
- 公式链路持续优化（渲染与兼容并重）

### 2) 专业级写作与排版体验

- 富文本智能粘贴为干净 Markdown
- 30 套主题 + 自定义风格
- 手机 / 平板 / 桌面多端预览
- 同步滚动与编辑效率优化

### 3) 多格式交付

- HTML / PDF / DOC / DOCX 导出
- 适配公众号、知识库、博客和办公文档流转

## 适合谁

- 长文作者与技术写作者
- 公众号内容团队与运营
- 需要高频发布的知识型创作者
- 对“最终发布质量”有明确要求的产品团队

## 为什么现在它已经像一个产品

因为它已经具备产品的核心特征：

- 明确场景（内容发布）
- 明确价值（发布稳定性与表达质量）
- 明确用户路径（写作到分发）
- 持续迭代的体验与工程闭环

它不再只是一个 demo，而是一条可持续演进的产品线。

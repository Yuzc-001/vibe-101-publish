# vibe-101-publish 宣传文案

> 把 Markdown 工具，从“编辑器”做成真正能发布的工作台。

---

## 一句话定位

`vibe-101-publish` 是面向公众号、知乎、Word 与社交平台的内容排版与发布工作台。  
它的重点不是“编辑区看起来好看”，而是“复制、导出、发布之后依然稳”。

在线体验：<https://yuzc-001.github.io/vibe-101-publish/>

## 当前产品口径

现在它已经不是单一的公众号工具，而是一条更完整的发布链路：

- 写作：富文本粘贴转 Markdown，快速进入编辑状态
- 排版：主题切换、多端预览、滚动同步、排版信息辅助检查
- 发布：复制到公众号、复制到知乎、复制到 Word
- 导出：PDF、HTML、`.docx`、`.doc`
- 分发：卡片工作台支持公众号长图与社交卡片导出

## 这轮升级最值得讲的点

### 1. 知乎复制链路重写

- 不再把预览 DOM 生硬塞进知乎
- 改成知乎专用剪贴板 HTML
- 公式、代码块、表格、链接等更稳定
- 浏览器不支持富文本复制时，还有 Markdown 导入兜底

### 2. 卡片功能升级为卡片工作台

- 一篇文章可按 `---` 拆成多张卡片
- 支持公众号长图和 `440 x 586` 社交卡片双模式
- 可以加封面、尾页和页码
- 可导出当前卡、全部 PNG，或直接打包 ZIP

### 3. 发布细节真正被打磨过

- 表格主渲染链修复，预览和复制更一致
- 图片坏链会被清理，不容易出现大白块
- 社交卡片会对图片和长内容做收敛
- 工具栏与卡片工作台都按实际使用习惯重新梳理

## 核心价值

| 维度 | 价值 |
| --- | --- |
| 发布稳定性 | 内容到平台后不容易塌样式 |
| 跨渠道能力 | 公众号、知乎、Word、社交卡片一套稿子走通 |
| 表达质量 | 主题、公式、代码、表格、图片都可控 |
| 工作流闭环 | 从写作到导出再到分发，不靠多工具拼接 |

## 适合人群

- 公众号作者与技术写作者
- 要把一份稿子分发到多个平台的内容团队
- 想把长文转成社交图片卡片的运营团队
- 对发布后稳定性有明确要求的产品或品牌团队
